# 自我进化学习笔记

## 1. Capability Evolver (能力进化引擎)
- 自动分析运行历史，识别改进点
- 自我修复：检测崩溃并建议补丁
- GEP协议：标准化可审计的进化
- 关键配置：
  - EVOLVE_ALLOW_SELF_MODIFY: 不建议开启，会导致不稳定
  - EVOLVE_STRATEGY: balanced/innovate/harden/repair-only等模式

## 2. Error Learning (错误学习)
- 从失败中提取教训
- 记录到 memory/错误日志.md
- 识别场景：
  - 任务执行3次以上仍失败
  - 工具调用返回错误
  - 用户反馈"不对"
  - 工具调用超时（超过30秒）

## 3. Self-Improvement (自我进化)
- 捕获并分析错误信息
- 从失败中提取教训
- 识别可复用工作模式
- 自动更新技能和记忆
- 超时自动恢复机制
- 周期性自检机制

## 4. 互联网学习（2026-02-24）

## 核心要点
