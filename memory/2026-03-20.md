- 逆向了人人租 OSS 上传 SDK（`rrzuOssSdkUmd.js` + `oss-sdk.js`）
- 完整上传链路：
  1. `GET https://go-micro.rrzu.com/file/oss/token?project=shop-admin&file_type=2` → AES 加密凭证
  2. SDK 用 cookie 密钥（`shop-admin_oss_token_key`）AES 解密
  3. 得到 `{host, policy, access_id, signature, dir, token, url}`
  4. `POST https://rrzu-backend.oss-cn-shanghai.aliyuncs.com` FormData 上传
  5. CDN 地址：`https://img1.rrzuji.cn/{path}`
- 实测上传成功：`https://img1.rrzuji.cn/shop-admin/images/20260320/3BAn98plhRyLfr5McUlvPwAzuAZ.jpg`
