# 人人租上架助手 🐂

> 状态说明（2026-03-15）：这是**内容与素材辅助项目**，不是自动上架主线。用于生成标题/描述/套餐/图片处理辅助，不负责真实后台提交流程。

快速生成商品信息的半自动化工具，帮你快速完成人人租商品上架。

## 功能特点

✅ **自动生成内容**
- 商品标题：品牌 + 型号 + 成色
- 商品描述：规范化格式，包含特点说明
- 租赁套餐：1/3/6/12月套餐自动计算
- 定价信息：押金、买断价、月租金

✅ **智能定价**
- 押金 = 闲鱼价 × 30%
- 买断价 = 闲鱼价 × 95%
- 月租金按周期递减（12%/10%/8%/6%）

✅ **图片处理**（可选）
- 上传商品图片
- 自动白底处理
- 一键下载成品

✅ **一键复制**
- 所有内容支持一键复制
- 直接粘贴到人人租后台

## 快速开始

### 1. 安装依赖

```bash
cd /Users/xs/.openclaw/workspace/projects/rrz-helper
npm install
```

### 2. 启动服务

```bash
npm start
```

服务将在 http://localhost:8067 启动

### 3. 使用流程

1. 打开浏览器访问 http://localhost:8067
2. 填写商品信息：
   - 品牌（如：Apple）
   - 型号（如：iPhone 15 Pro Max 256GB）
   - 成色（全新/99新/95新等）
   - 闲鱼参考价（元）
   - 商品特点（可选）
3. 点击"生成内容"
4. 复制生成的标题、描述、套餐信息
5. 粘贴到人人租后台对应位置

### 4. 图片处理（可选）

1. 点击上传区域或拖拽图片
2. 支持多张图片批量上传
3. 点击"处理图片"
4. 自动下载白底处理后的图片

## 定价规则

当前定价规则（可在 script.js 中调整）：

```javascript
押金 = 闲鱼价 × 30%
买断价 = 闲鱼价 × 95%

月租金：
- 1个月：闲鱼价 × 12%
- 3个月：闲鱼价 × 10%

- 12个月：闲鱼价 × 6%
```

## 注意事项

1. **图片处理功能**需要 Node.js 服务运行，如果只需要文本生成，可以直接打开 index.html
2. **定价规则**可根据实际情况在 script.js 中调整
3. **商品描述**模板可在 script.js 的 generateContent() 函数中自定义
4. 建议使用 Chrome/Edge 浏览器以获得最佳体验

## 常见问题

**Q: 图片处理失败？**
A: 确保已运行 `npm install` 安装依赖，并且服务已启动（npm start）

**Q: 如何修改定价规则？**
A: 编辑 script.js 文件顶部的 PRICING_RULES 对象

**Q: 可以离线使用吗？**
A: 文本生成功能可以离线使用（直接打开 index.html），图片处理需要本地服务
