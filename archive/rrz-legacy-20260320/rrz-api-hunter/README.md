# 人人租API逆向工程

> 状态说明（2026-03-15）：这是**辅助侦查项目**，用于抓包、逆向、找上传/提交接口，不是当前默认主线。只有在浏览器链路卡住、需要找 API 旁路时再进入。

## 目标
用纯requests实现商品发布，不依赖浏览器自动化

## 快速开始

### 方法1：浏览器控制台抓包（推荐⭐）

1. 打开 https://admin.rrzu.com 并登录
2. 按F12，切换到Console标签
3. 复制并运行 `api_interceptor.js` 的内容
4. 操作页面（点击商品列表、发布商品、上传图片等）
5. 运行 `exportApiLog()` 下载日志
6. 将日志文件放到 `captured/` 目录

### 方法2：Playwright自动抓包

```bash
python3 api_hunter.py
# 在浏览器中操作，按Ctrl+C停止
```
