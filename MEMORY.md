- 遇到网络超时/工具卡死，设置30秒超时，超时立即放弃

| 白底处理 | ⚠️ | PIL叠加，基础可用 |
| AI审核 | ✅ | 中转API+base64 |

### 技术要点
- **火山引擎限制**：无法直接访问外网图片URL
- **解决方案**：用中转API + base64本地图片
- **中转API**：https://www.zhongzhuan.win/v1 + sk-5e5rZgZ17IiakvNWPDFCKfdSzhlvQ1AlZtv5zfnPcE9FUuYk
- **抠图工具**：rembg（需先pip install rembg）

### 白底处理流程
```python
from rembg import remove
from PIL import Image

# 1. 抠图
img = Image.open('input.jpg')
output = remove(img)
output.save('rmbg.png')

# 2. 叠加白底
white_bg = Image.new('RGBA', (800, 800), (255, 255, 255, 255))
white_bg.paste(output, ((800-output.width)//2, (800-output.height)//2))
white_bg.convert('RGB').save('output.jpg', 'JPEG', quality=95)
```
