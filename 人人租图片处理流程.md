# 人人租商品图片获取处理流程

> 从原始图片到符合平台要求的白底主图 🖼️

---

## 📋 流程总览

```
图片获取 → AI初筛 → 白底处理 → AI审核 → 命名保存
```

**预计耗时：** 单个商品 5-10 分钟

---

## 🎯 第1步：图片获取

### 方式1：官网/电商平台爬取

**适用场景：** 有官方商品页面

**工具：** `skills/image-crawler/image_crawler.py`

```bash
python3 skills/image-crawler/image_crawler.py <商品页面URL>
```

**流程：**
1. 浏览器访问商品页面
2. JS提取所有图片URL（优先 alicdn.com 等CDN域名）
3. 下载原始图片到本地
4. 保存到 `downloads/` 目录

**注意事项：**
- 必须用已登录的浏览器（避免反爬）
- 提取原始图片URL，不要截图
- 过滤掉UI元素、按钮、图标等

---

### 方式2：用户提供图片

**适用场景：** 用户直接发送商品照片

**处理：**
1. 保存到工作目录
2. 检查图片尺寸（建议 ≥600×600）
3. 如果是实拍图，可能需要抠图

## 🎨 第3步：白底处理

**目标：** 纯白色背景 RGB(255,255,255)

### 方法1：抠图 + 白底（推荐）

**工具：** rembg

```bash
pip install rembg
```

**代码：**
```python
from rembg import remove
from PIL import Image

# 1. 抠图
img = Image.open('input.jpg')
output = remove(img)  # 背景透明
output.save('rmbg.png')

# 2. 叠加白底
white_bg = Image.new('RGBA', (800, 800), (255, 255, 255, 255))
# 居中放置
x = (800 - output.width) // 2
y = (800 - output.height) // 2
white_bg.paste(output, (x, y), output)  # 第三个参数是mask

# 3. 转换为RGB保存
final = white_bg.convert('RGB')
final.save('output.png', 'PNG')
```

---

### 方法2：阈值替换（快速但不精确）

**适用场景：** 原图背景已经接近白色

**代码：**
```python
from PIL import Image
import numpy as np

img = Image.open('input.jpg')
arr = np.array(img)

# 阈值>200，覆盖浅灰色区域
mask = np.all(arr[:,:,:3] > 200, axis=2)
arr[mask] = [255, 255, 255]

out = Image.fromarray(arr)
out.save('output.png', 'PNG')
```

### 尺寸调整

**目标尺寸：** 800×800px（推荐）

```python
from PIL import Image

img = Image.open('input.png')

# 等比缩放到800×800内
img.thumbnail((800, 800), Image.Resampling.LANCZOS)

# 创建800×800白底画布
canvas = Image.new('RGB', (800, 800), (255, 255, 255))

# 居中粘贴
x = (800 - img.width) // 2
y = (800 - img.height) // 2
canvas.paste(img, (x, y))

canvas.save('output.png', 'PNG')
```

## ⚠️ 注意事项

### 图片格式

- **推荐：** PNG（无损，避免JPEG压缩灰化）
- **备选：** JPEG quality=100

### 白底验证

- 上传到人人租后，必须下载服务器实际图片验证
- 人人租服务器会把PNG转JPG，可能导致灰化
- 如发现灰化，降低阈值（>200）重新处理

### 文字过滤

- 商品自身的品牌Logo、型号刻字**不算违规**
- 商家添加的水印、联系方式**必须去除**

## 🛠️ 完整自动化脚本

```python
#!/usr/bin/env python3
"""
人人租商品图片处理流水线
"""

import os
import base64
import requests
from PIL import Image
from rembg import remove
import numpy as np

API_BASE = 'https://www.zhongzhuan.win/v1'
API_KEY = 'sk-5e5rZgZ17IiakvNWPDFCKfdSzhlvQ1AlZtv5zfnPcE9FUuYk'
MODEL = 'aws.amazon/claude-opus-4-5:once'

def ai_call(prompt, image_path):
    """调用AI API"""
    with open(image_path, 'rb') as f:
        img_b64 = base64.b64encode(f.read()).decode()
    
    response = requests.post(
        f'{API_BASE}/chat/completions',
        headers={'Authorization': f'Bearer {API_KEY}'},
        json={
            'model': MODEL,
            'messages': [{
                'role': 'user',
                'content': [
                    {'type': 'text', 'text': prompt},
                    {'type': 'image_url', 'image_url': {'url': f'data:image/jpeg;base64,{img_b64}'}}
                ]
            }]
        }
    )
    return response.json()['choices'][0]['message']['content']

def filter_image(image_path):
    """AI初筛：是否为商品主图"""
    answer = ai_call('这是商品实物主图吗？只回答"是"或"否"', image_path)
    return '是' in answer

def process_white_bg(image_path, output_path):
    """白底处理：抠图 + 白底 + 居中"""
    # 1. 抠图
    img = Image.open(image_path)
    output = remove(img)
    
    # 2. 叠加白底
    canvas = Image.new('RGBA', (800, 800), (255, 255, 255, 255))
    
    # 等比缩放
    output.thumbnail((700, 700), Image.Resampling.LANCZOS)
    
    # 居中
    x = (800 - output.width) // 2
    y = (800 - output.height) // 2
    canvas.paste(output, (x, y), output)
    
    # 3. 保存
    final = canvas.convert('RGB')
    final.save(output_path, 'PNG')

def audit_image(image_path):
    """AI审核：检查是否符合要求"""
    prompt = '''检查这张商品图：
1. 背景是否纯白？
2. 是否有商家水印/联系方式？（商品自身的品牌Logo/型号刻字不算）
3. 图片是否清晰？
4. 商品是否居中完整？

只回答"通过"或列出问题。'''
    
    answer = ai_call(prompt, image_path)
    return '通过' in answer, answer

def process_product_images(input_dir, brand, model):
    """处理商品图片流水线"""
    output_dir = f'products/{brand}/{model}'
    os.makedirs(output_dir, exist_ok=True)
    
    images = [f for f in os.listdir(input_dir) if f.endswith(('.jpg', '.png', '.jpeg'))]
    
    passed = []
    for i, img_file in enumerate(images, 1):
        input_path = os.path.join(input_dir, img_file)
        print(f'处理 {img_file}...')
        
        # 1. AI初筛
        if not filter_image(input_path):
            print(f'  ❌ 不是商品主图，跳过')
            continue
        
        # 2. 白底处理
        temp_path = f'/tmp/temp_{i}.png'
        process_white_bg(input_path, temp_path)
        print(f'  ✅ 白底处理完成')
        
        # 3. AI审核
        ok, msg = audit_image(temp_path)
        if not ok:
            print(f'  ❌ 审核不通过: {msg}')
            continue
        
        # 4. 保存
        output_path = os.path.join(output_dir, f'{brand}_{model}_{i:02d}.png')
        os.rename(temp_path, output_path)
        passed.append(output_path)
        print(f'  ✅ 保存到 {output_path}')
    
    print(f'\n完成！通过 {len(passed)}/{len(images)} 张图片')
    return passed

if __name__ == '__main__':
    import sys
    if len(sys.argv) != 4:
        print('用法: python3 process_images.py <输入目录> <品牌> <型号>')
        sys.exit(1)
    
    input_dir, brand, model = sys.argv[1:4]
    process_product_images(input_dir, brand, model)
```

**使用方法：**
```bash
python3 process_images.py ./downloads Apple MacMini_M4
```
