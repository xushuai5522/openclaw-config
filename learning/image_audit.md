# AI图片审核系统

用于电商上架前的图片质量检测与筛选。

## 审核维度

### 1. 基础质量
- 分辨率 >= 600x600px
- 文件大小 > 50KB
- 格式支持: JPG, PNG, WebP

### 2. 图片内容
- 产品主体清晰可见
- 无严重模糊或失焦
- 无明显噪点
- **仅商品主体照片，不能有文字**
- 无多余背景物体

### 3. 背景检查
- 白底/纯色底优先
- 非白底需可去除背景

### 4. 水印检测
- 检测右下角水印
- 检测大面积logo/文字遮挡
- 检测半透明水印

### 5. 版权风险
- 检测知名品牌logo（如Apple）
- 第三方平台水印（淘宝、京东等）

### 6. 合规检查
- 无违规内容（色情、暴力等）
- 无虚假宣传

## 审核结果

| 等级 | 说明 | 建议 |
|------|------|------|
| ✅ A级 | 完美，可直接使用 | 通过 |
| ⚠️ B级 | 需要简单处理 | 裁剪/去水印后使用 |
| ❌ C级 | 不适合使用 | 丢弃 |

## 使用流程

```
爬取图片 → AI检测 → 分类处理 → 上架使用
```

## 实现方案

使用Python进行图片分析：
- Pillow: 基础质量检测
- 颜色分析: 背景检测
- 模板匹配: 水印检测
- 机器学习: 内容分类（可选）
