# 人人租商品上架流程学习笔记

## 演示概况
- 时间：2026-02-20
- 商品：苹果 Mac studio M4 Max 2025款
- 录屏：600张截图 (0.5秒/张)

---

## 上架流程关键步骤

### 1. 进入发布页面
- 后台地址：https://admin.rrzu.com
- 路径：店铺 → 商品管理 → 发布商品

### 2. 基础信息填写
| 字段 | 示例值 | 备注 |
|------|--------|------|
| 商品标题 | 90新 苹果 Mac studio M4 Max 2025款 | 必须包含"90新"等成新描述 |
| 商品类型 | 二手 | 必选 |
| 设备成新 | 90新 | 必选 |

### 3. 类目属性（系统自动识别）
| 字段 | 示例值 |
|------|--------|
| 品牌 | 苹果 |
| 型号 | Mac studio（M4 Max）2025款 |
| CPU | M4 Max |
| 运行内存 | 其他 |
| 硬盘容量 | 2TB |
| 显卡类型 | 集显 |

### 4. 销售信息
- 定价模式：默认模式(2.0)
- 租赁方案：
  - 到期须归还：勾选
  - 最低租期：1个月
  - 最高租期：1年
  - 租期配置：1个月、3个月、6个月、1年

### 5. 销售规格（表格填写）
| 字段 | 示例值 |
|------|--------|
| 押金 | 25000 |
| 库存 | 5 |
| 1个月租金 | 2500 |
| 3个月租金 | 7500 |
| 6个月租金 | 14000 |
| 1年租金 | 25000 |

### 6. 商品信息
- 商品图片：需上传（600x600px以上，纯白底）
- 注意事项：可勾选支持开发票、租用服务设备等
- 服务标签：缺货必赔(默认)、包邮配送、隐私清除、90天质保、租期质保
- 商品描述：必填

### 7. 支付信息
- 商品上架端口：个人 + 码商（默认都勾选）

### 8. 物流信息
| 字段 | 示例值 |
|------|--------|
| 发货地址 | 江苏省苏州市...（已配置） |
| 归还地址 | 江苏省苏州市...（已配置） |
| 发货时效 | 16:00 |
| 寄出快递 | 顺丰快递 |
| 配送方式(寄出) | 包邮 |
| 配送方式(归还) | 包邮 |

---

## 审核不通过常见原因（需避免）
1. 标题必须包含设备成新描述，如"90新"
2. 套餐内需要详细写明具体租用方式
3. 删除套餐中的"租赁"字眼

---

## 自动化难点
1. 图片上传：需要从素材中心选择或本地上传
2. 下拉选择：需要点击展开后选择选项
3. 销售规格表格：需要填写多个租期的价格
