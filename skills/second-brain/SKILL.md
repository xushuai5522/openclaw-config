# Second Brain - 第二大脑

## 技能说明

将对话和工作中产生的知识自动沉淀为持久记忆，建立个人知识库。类似"第二大脑"，帮助AI记住学过的东西。

## 使用场景

1. **大哥要求"记住这个"**：明确让AI记住某个知识
2. **遇到有价值的信息**：发现值得沉淀的内容
3. **需要查询过去**：大哥问"之前那个xxx呢？"
4. **构建知识体系**：系统化整理某个领域的知识

## 执行步骤

### 第一步：识别知识

识别以下类型的内容：
- 大哥明确说"记住..."
- 重要的决定或偏好
- 学到的新技能/工具
- 有用的信息/技巧
- 工作流程/最佳实践

### 第二步：存储知识

根据内容类型选择存储位置：
| 内容类型 | 存储位置 |
|----------|----------|
| 用户偏好 | USER.md |
| 工作原则 | AGENTS.md |
| 工具配置 | TOOLS.md |
| 重要记忆 | MEMORY.md |
| 日常记录 | memory/YYYY-MM-DD.md |

### 第三步：知识检索

当大哥问起时：
1. 先搜索 MEMORY.md
2. 再搜索 memory/ 目录
3. 找到后提取相关信息
4. 以自然的方式回复

### 第四步：知识关联

将新知识与已有知识关联：
- 找到相关主题
- 在对应章节添加链接或备注
- 保持知识结构化

## 知识结构

建议的知识库结构：
```
MEMORY.md
├── 用户偏好
├── 工作原则
├── 技能/工具
├── 重要记忆
└── 教训总结

memory/
├── 2026-01-01.md
├── 2026-01-02.md
└── ...
```

## 注意事项

- 不要什么都要记，只记有价值的
- 定期清理重复/过时的内容
- 保持知识库精简
- 尊重隐私，不过度记录

## 成功标准

- 知识能有效沉淀
- 需要时能快速检索
- 知识结构清晰
- 大哥满意
