# 💰 家庭理财小管家

## 快速参考
- **触发词**: 记账、花了多少钱、支出、预算、账单
- **核心功能**: 主动记账、智能分析、预算管理
- **服务地址**: http://localhost:8066

---

主动记账 + 智能分析的家庭财务管理助手。

## 核心功能

### 1. 主动记账
- 从对话中自动识别消费行为（"刚买了杯咖啡 30块"、"交了房租 3000"）
- 支持手动记账：`记账 午餐 35`
- 支持批量导入

### 2. 分类体系
| 大类 | 子类示例 |
|------|----------|
| 🍜 餐饮 | 早餐、午餐、晚餐、外卖、饮品 |
| 🏠 居住 | 房租、水电、物业、维修 |
| 🚗 交通 | 打车、地铁、加油、停车 |
| 🛒 购物 | 日用品、衣服、电子产品 |
| 🎮 娱乐 | 游戏、电影、订阅服务 |
| 💊 医疗 | 看病、药品、保险 |
| 📚 教育 | 课程、书籍、培训 |
| 💼 工作 | 办公用品、工具、服务器 |
| 🎁 人情 | 红包、礼物、请客 |
| 📦 其他 | 未分类 |

### 3. 智能分析
- 每周/每月支出报告
- 异常消费提醒（单笔超过日均3倍）
- 分类占比分析
- 同比/环比趋势

### 4. 预算管理
- 设置月度总预算和分类预算
- 预算使用进度提醒（50%/80%/100%）
- 超支预警

## 数据存储

所有数据存储在飞书多维表格中：
- 表1：账单明细（日期、金额、分类、子类、备注、来源）
- 表2：预算设置（分类、月预算、生效月份）
- 表3：月度汇总（月份、分类、总支出、预算、差额）

## 记账格式

### 快速记账
```
记账 [描述] [金额]
记账 午餐 外卖 35
记账 打车回家 28.5
```

### 收入记录
```
收入 [描述] [金额]
收入 工资 15000
收入 闲鱼卖货 200
```

### 查询
```
本月支出          → 本月总支出 + 分类明细
本周花了多少      → 本周支出汇总
餐饮花了多少      → 指定分类支出
最近10笔          → 最近记录
```

## 主动记账规则

当检测到对话中包含消费相关信息时：
1. 提取金额和描述
2. 自动分类
3. 确认后记录（首次使用需确认，熟悉后可自动记录）
4. 回复格式：`✅ 已记账：[分类emoji] [描述] ¥[金额]`

## 飞书多维表格结构

### 账单明细表字段
| 字段名 | 类型 | 说明 |
|--------|------|------|
| 日期 | 日期 | 记账日期 |
| 类型 | 单选 | 支出/收入 |
| 大类 | 单选 | 餐饮/居住/交通等 |
| 子类 | 文本 | 具体子分类 |
| 金额 | 数字 | 金额（元） |
| 备注 | 文本 | 描述信息 |
| 来源 | 单选 | 手动/自动/导入 |

### 预算表字段
| 字段名 | 类型 | 说明 |
|--------|------|------|
| 分类 | 单选 | 对应大类 |
| 月预算 | 数字 | 预算金额 |
| 生效月份 | 文本 | YYYY-MM |

## 使用流程

### 首次使用
1. 创建飞书多维表格（自动）
2. 设置月度预算（可选）
3. 开始记账

### 日常使用
- 对话中提到消费 → 自动识别并记账
- 手动记账 → `记账 xxx xx`
- 查看报告 → `本月支出` / `周报`
