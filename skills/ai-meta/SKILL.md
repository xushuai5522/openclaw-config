# ai-meta - AI元能力工具箱

AI自我进化、学习、记忆管理的综合工具。

## 功能模块

### 1. 能力进化 (capability-evolver)
- 运行时历史分析
- 识别改进点
- 协议约束进化
- 自动应用优化

### 2. 错误学习 (error-learning)
- 从失败中提取教训
- 错误模式识别
- 避免重复踩坑
- 建立错误知识库

### 3. 自我改进 (self-improvement)
- 扫描聊天记录
- 捕获错误模式
- 提取可复用技能
- 知识库升级

### 4. 第二大脑 (second-brain)
- 对话转化为持久知识
- 个人知识库管理
- 语义搜索
- 知识图谱构建

### 5. 每日复盘 (daily-review)
- 自动从对话提取有价值信息
- 沉淀到长期记忆
- 生成每日总结
- 维护MEMORY.md

## 快速参考

- **触发词**: 错误学习、复盘、记忆维护、能力进化
- **核心功能**: 自动学习、错误记录、知识沉淀
- **输出文件**: MEMORY.md, memory/YYYY-MM-DD.md, errors.json

## 使用示例

### 示例1: 错误学习 - 浏览器代理问题

**场景**: 浏览器连接失败，发现是代理配置问题

```bash
# 1. 记录错误
echo '{
  "error": "浏览器代理连接失败",
  "cause": "系统代理配置了127.0.0.1:51362但代理服务未启动",
  "solution": "关闭系统代理或启动代理服务",
  "prevention": "启动浏览器前先检查代理端口连通性",
  "timestamp": "2026-03-02",
  "count": 3
}' >> ~/.openclaw/workspace/memory/errors.json

# 2. 更新MEMORY.md
echo "## 浏览器代理问题（已解决）
- 问题：代理配置但服务未启动
- 解决：networksetup关闭系统代理
- 预防：启动前nc -zv检查端口
" >> ~/.openclaw/workspace/MEMORY.md
```

**效果**: 下次遇到同样问题，AI会自动想起解决方案，不再重复犯错。

### 示例2: 能力进化 - 优化图片处理流程

**场景**: 发现图片处理重复执行多个步骤，效率低

```bash
# 1. 识别低效模式
# 原流程：爬图 → 下载 → AI筛选 → 白底处理 → AI审核
# 问题：每步都是独立命令，需要人工确认

# 2. 提出改进方案
# 新流程：一条命令完成全流程
python image_pipeline.py --keyword "小米平板5" --full-pipeline

# 3. 记录改进
echo '{
  "improvement": "图片处理流水线化",
  "before": "5个独立步骤，需要人工确认",
  "after": "1条命令自动完成",
  "efficiency": "时间节省80%",
  "timestamp": "2026-03-02"
}' >> ~/.openclaw/workspace/memory/improvements.json
```

**效果**: 工作流程自动优化，效率大幅提升。

## 使用场景

### 能力进化
当发现重复出现的问题或低效模式时，自动分析并提出改进方案。

### 错误学习
每次遇到错误后，记录错误原因、解决方案、预防措施，避免重复犯错。

### 自我改进
定期扫描历史对话，提取可复用的技能和知识，更新到技能库。

### 第二大脑
将重要的对话、决策、想法转化为结构化知识，方便后续检索和使用。

### 每日复盘
每天自动总结当天的工作、学习、决策，沉淀到长期记忆。

## 工作流程

```
对话/操作 → 实时记录 → 错误捕获 → 模式识别 → 知识提取 → 能力进化
    ↓           ↓           ↓           ↓           ↓           ↓
 原始数据    日志文件    错误库    改进建议    知识库    技能升级
```

## 核心原则

1. **持续学习** - 从每次交互中学习
2. **避免重复** - 记录错误，不再犯同样的错
3. **知识沉淀** - 将经验转化为可复用的知识
4. **自动进化** - 自动识别改进点并应用

## 整合说明

本技能整合了以下原有技能：
- capability-evolver - 能力进化引擎
- error-learning - 错误学习系统
- self-improvement - 自我改进机制
- second-brain - 第二大脑知识库
- daily-review - 每日复盘工具

所有AI元能力统一在此技能中管理。

## 配置文件

- `memory/` - 日常记忆文件
- `MEMORY.md` - 长期记忆
- `AGENTS.md` - 行为规则
- `errors.json` - 错误知识库
- `improvements.json` - 改进记录
