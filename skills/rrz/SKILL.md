# 人人租运营工具箱

> 职责：后台基础操作（登录、弹窗、审核修复、列表查询）。
> 发布规则/定价 → `rrz-publish`；完整上架流程 → `rrz-e2e`。

## 基础信息

| 项目 | 值 |
|------|-----|
| 后台地址 | https://admin.rrzu.com |
| 商品列表 | https://admin.rrzu.com/spu-view/list |
| 发布页 | https://admin.rrzu.com/spu-view/create |
| 编辑页 | https://admin.rrzu.com/spu-view/create?id={id} |
| API 基础 | https://go-micro.rrzu.com |
| iframe | name=rrzuji, 域名 admin-vue.rrzu.com |
| 店铺 | 书电马数码店 |

## 页面结构

- 主页面 `admin.rrzu.com` + iframe `admin-vue.rrzu.com`（跨域）
- browser snapshot 自动穿透 iframe
- JS 注入需通过 `frame: "rrzuji"` 参数

## 弹窗处理

每次进入后台先关弹窗：
1. 经营地址信息补充 → 点"跳过"
2. 方案应用邀请 → 点关闭（不点"确定"，会清空字段）
3. 工单提醒 → 点"×"
4. 通知面板 → 点 Close

## 登录恢复

遇到登录页自行恢复，不求助大哥。

1. 访问 `https://www.rrzu.com/auth/server-login`
2. 切到"密码登录"tab
3. 输入手机号 + 密码
4. 勾选隐私协议
5. JS 点击登录按钮
6. 等待跳转到 `admin.rrzu.com`

## 审核规则速查

### 标题
- 格式：`{成色} {品牌} {型号} {使用场景}`
- 必须含具体型号，场景用动词短语（不用形容词）

### 禁止词
租赁→租用、出租→删除、免押/免息/分期/最→删除、测试→调试

### 套餐
- 名称必须是租用方式：租完归还 / 到期可归还/续租
- 规格表头用"型号/配置/参数"，不用"套餐"

### 图片
- 白底 #FFF，≥600×600（推荐800×800）
- 主图 + 描述图都必须有

## 审核修复流程

1. 商品列表找到"审核不通过"商品
2. 点"修改商品"进入编辑页
3. 按不通过原因修复（标题/套餐名/禁止词/规格表头/描述）
4. 提交审核
5. 回列表确认状态变化（不信 toast）

### 常见不通过原因

| 原因 | 修复 |
|------|------|
| 标题缺型号 | 添加完整型号 |
| 套餐名缺租用方式 | 改为"租完归还"等 |
| 含"租赁"字眼 | 替换为"租用" |
| 规格表头有误 | "套餐"→"型号/配置" |
| 标题与描述型号不一致 | 统一 |
| 需增加套餐规格 | 添加更多租期 |
| 商品描述无图片 | 上传描述图 |

## 防重复创建

1. 新建前先搜索列表查重
2. `/spu-view/create` 无 id = 新建；有 id = 编辑
3. 提交后必须回列表拿到商品 ID + 状态变化
4. 状态未知时禁止二次新建
