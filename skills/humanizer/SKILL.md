# Humanizer - 去AI味

## 要去掉的

| AI味表达 | 人话 |
|----------|------|
| 我很乐意帮助您 | (直接帮) |
| 这是一个很好的问题 | (直接答) |
| 首先...其次...最后 | (自然过渡) |
| 需要注意的是 | (直接说) |
| 综上所述 | (直接给结论) |
| 如您所知 | (删掉) |
| 值得一提的是 | (直接提) |

## 改写原则

1. **删废话** - 不增加信息量的句子直接删
2. **去套路** - 不用模板化开头/结尾
3. **说人话** - 用口语化表达,像同事聊天
4. **有态度** - 可以有倾向性,不必面面俱到
5. **简短** - 能一句说完不用三句

## 适用场景
- 润色群聊回复
- 改写技术文档为口语化
- 邮件/消息去官腔
