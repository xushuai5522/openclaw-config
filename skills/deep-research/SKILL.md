# Deep Research

## 快速参考
- **触发词**: 深度研究、调研、分析报告
- **核心功能**: 多步推理、信息整合、报告生成
- **适用场景**: 技术调研、市场分析、竞品研究

## 使用示例

### 技术调研
```
用户：研究一下2026年AI Agent的发展趋势

AI执行流程：
1. 搜索最新资料（学术论文、行业报告、技术博客）
2. 分析主流观点（OpenAI、Anthropic、Google等）
3. 整合关键信息（技术突破、应用场景、挑战）
4. 生成调研报告（趋势总结、预测、建议）
```

### 市场分析
```
用户：分析小红书电商市场现状

AI执行流程：
1. 收集数据（用户规模、GMV、品类分布）
2. 竞品对比（抖音、淘宝、拼多多）
3. 用户画像（年龄、性别、消费习惯）
4. 生成报告（机会点、风险、策略建议）
```

### 竞品研究
```
用户：对比OpenClaw和其他AI Agent框架

AI执行流程：
1. 列出主要竞品（AutoGPT、LangChain、CrewAI）
2. 功能对比（核心能力、易用性、生态）
3. 优劣分析（各自优势、适用场景）
4. 选型建议（根据需求推荐）
```

## 使用场景
- 技术选型前的调研
- 市场进入前的分析
- 竞品策略研究
- 行业趋势预测
