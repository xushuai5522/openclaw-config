# 人人租运营管理协同工作计划

## 目标
完成人人租机运营管理功能，实现商品发布、审核修复、状态监控的完整自动化。

---

## 设备分工

### Mac（电子牛马）
- **角色**: 主控协调 + 图片处理
- **负责**:
  - 商品信息收集和整理
  - 图片爬取、白底处理、AI审核
  - 定价计算
  - 协调NEC设备执行任务
  - 监控整体进度

### NEC（192.168.1.17）
- **角色**: 浏览器自动化执行
- **负责**:
  - 人人租后台登录保持
  - 商品发布表单填写
  - 审核不通过商品修复
  - 商品状态查询
  - 执行结果反馈

---

## 核心功能模块

### 1. 商品发布流程
```
Mac: 收集信息 → 处理图片 → 计算定价
  ↓
NEC: 登录后台 → 填写表单 → 上传图片 → 提交审核
  ↓
Mac: 验证提交结果 → 记录商品ID
```

### 2. 审核修复流程
```
NEC: 查询审核不通过商品 → 获取拒绝原因
  ↓
Mac: 分析原因 → 生成修复方案
  ↓
NEC: 执行修复 → 重新提交
  ↓
Mac: 验证修复结果
```

### 3. 状态监控流程
```
NEC: 定期查询商品状态（每2小时）
  ↓
Mac: 汇总统计 → 发现问题 → 触发修复流程
```

## 执行原则

1. **先验证再执行** - 每个操作前先截图确认状态
2. **及时反馈** - 执行过程中实时报告进度
3. **记录日志** - 所有操作记录到日志文件
4. **互相监督** - Mac监控NEC执行，NEC反馈执行结果
5. **失败重试** - 遇到错误最多重试3次，失败后人工介入
